    ========================================
    【使用教程】
    ========================================
    使用说明：
1. 安装 Python 3.10+
2. 安装 PyInstaller: pip install pyinstaller
3. 运行本工具打包你的 Python 程序

【基础使用】
1. 选择Python文件：点击「选择」按钮或直接拖拽.py文件
2. 设置输出目录和文件名
3. 选择打包器（PyInstaller / Nuitka 等）
4. 点击「开始打包」

【高级设置】
• 排除模块：排除不需要的库以减小体积
• 隐藏导入：手动添加未能自动检测的模块
• 数据文件：添加配置文件、图片等资源文件
• 虚拟环境：创建项目独立环境

【常见问题】
Q: 打包后程序无法运行？
A: 检查是否缺少隐藏导入，可在「高级设置」中分析依赖并添加。

Q: 打包体积太大？
A: 使用「排除模块」功能排除不需要的库，或使用 Nuitka 编译。
